New Questions

Heads up: This page is still under construction! Guidelines and layout of this page may change rapidly

Thanks for your interest in submitting new questions! By adding new questions, you help keep trivia fun and exciting.

Guidelines

To help make sure questions are fun for everyone, there are a few guidelines that should be followed when new questions are submitted:

  • Questions should have a category. Sub-categories are not necessary but are allowed. We would prefer that the primary category be one of the below if possible
  • The fewer the words in an answer, the better. Questions should not be a guessing game to figure out the wording.
  • We have enough questions about Canada already.
  • Questions should not be multiple choice.
  • Questions must be verifiable and should be objective. We want trivia to be as correct as possible
  • Questions will ideally be unambiguous without hints. Please avoid questions like "Name that dog breed: ______"

Style guide for questions and tips

  • Questions with fill-in-the-blanks should use the standard of five underscores, "_____" (without quotations)
  • Questions will automatically have question marks added to the end. Submitting a new question with a question mark on the end will result in double question marks

Additional considerations

  • Questions about a specific fact about a specific year may be too narrow in scope or difficult
  • KAOS questions are particularly sensitive to long answers. Ideally, all KAOS answers will be 1 or 2 words
