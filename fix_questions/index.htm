Fix Questions

Fixing questions is a very simple process, and we would greatly appreciate it if you would like to help out. It doesn’t take long and everyone benefits from each question fixed.

How it works

First, register on Glime. Then, follow the instructions below to edit a question. You may edit any questions, even questions that haven’t been reported.

Please note the differences in numbers. There is the round number, round number, question number, edit number, and a temporary number. You will only need to know the round number, report number, and question number. Round number is the number displayed next to the question in the channel. The questions are selected at random from the database, and then assigned a round number for tracking. The question number is the ID for the question in the database. The report number is just the number for the reports, it does not correspond with a question directly. When reporting questions, you will need to know the round number. When fixing questions, you may need the report number, but you will always need the question number. The edit number and the temporary number are only used those approving edits and new questions.

Editing Reported Questions

Questions reported by trivia players will need to be fixed. You will need to message Glime with the listreports command. Make sure to include the channel. It should look like this: /msg Glime listreports #trivialand. To view more details about the report, use the showreport command. You will need to specify the channel number and the report number. Once you find a question you’d like to edit, you can use the edit command. You will need to specify the question number and the corrected version of the question (along with the answer). Now, you will need to remove the report with the removereport command. It will look like this: /msg Glime removereport #trivialand 151. Then you’re done!

Editing Question Without Reporting

You don’t need to report a question to edit it. There are two ways to fix a question this way. You can do this by reporting with regex. By reporting with regex, Glime will automatically make the changes to the question and submit it for approval immediately. You will need to know regex to use this reporting method. When you report the question, include the regex after the round number. It should look like this: .report 94854 s/cheese/bacon/ Glime will send you two messages, and reply back in the channel noting that regex was used.

That’s all you need to do. It is much simpler and quicker to use regex than to report without it, so please try to use this method as much as possible (so long as you know how to use it).

The other way is a bit more complicated, and I will get into details on how to do that at a later time.

Thank-you for the help! It means a lot to all of us. Those who’ve edited at least 100 questions are eligible for an editor cloak. In future versions of TriviaTime, you will also receive a badge!
