All About Questions

Please note the differences in numbers. There is the round number, report number, question number, edit number, delete number, and a temporary number. You will only need to know the round number, report number, and question number. Round number is the number displayed next to the question in the channel. The questions are selected at random from the database, and then assigned a round number for tracking. The question number is the ID for the question in the database. The report number is just the number for the reports, it does not correspond with a question directly. When reporting questions, you will need to know the round number. When fixing questions, you may need the report number, but you will always need the question number. The edit number and the temporary number are only used those approving edits and new questions.

We hold our questions to a high standard. When you are adding or editing questions, please follow the guidelines below to ensure your edit or addition it approved.

  • Questions should have a category. Sub-categories are not necessary.
  • Questions should not be easily guessed by the hints given. Think about how the answer will display in the hints for the question, and adjust accordingly.
  • The fewer the words in an answer, the better.
  • Questions should not become a guessing game. Avoid multiple choice (such as colors, dates, years, etc).
  • We have enough questions about Canada already.
  • Proposed questions should be appropriate for all audiences and interesting.
  • Avoid long questions.
  • Each question should be verifiable. There should be evidence for the question online.
  • Questions with fill-in-the-blanks should use the standard of five underscores, "_____" (without quotations).
  • KAOS need to have at least 10 answers, preferably only 10.

If you see a question that is not up to par to our expectations, please either report it, edit it, or delete it (if it can't be salvaged by editing).

Fix Questions

Fixing questions is a very simple process, and we would greatly appreciate it if you would like to help out. It doesn’t take long and everyone benefits from each question fixed.

How it works

First, register on Glime. Then, follow the instructions below to edit a question. You may edit any questions, even questions that haven’t been reported.

Editing Reported Questions

Questions reported by trivia players will need to be fixed. You will need to message Glime with the listreports command. Make sure to include the channel. It should look like this: /msg Glime listreports #trivialand. To view more details about the report, use the showreport command. You will need to specify the channel number and the report number. Once you find a question you’d like to edit, you can use the edit command. You will need to specify the question number and the corrected version of the question (along with the answer). Once the edit is accepted, the edited question will replace the former question, and the report will be removed.

Editing Question Without Reporting

You don’t need to report a question to edit it. There are two ways to fix a question this way. You can do this by reporting with regex. By reporting with regex, Glime will automatically make the changes to the question and submit it for approval immediately. You will need to know regex to use this reporting method. When you report the question, include the regex after the round number. It should look like this: .report 94854 s/cheese/bacon/ Glime will send you two messages, and reply back in the channel noting that regex was used.

That’s all you need to do. It is much simpler and quicker to use regex than to report without it, so please try to use this method as much as possible (so long as you know how to use it).

Add Questions

Ready to make trivia more interesting? By submitting your questions, not only will the overall question quality improve, but you will have an advantage since you already know the answers! Read below on how to submit your questions.

Message Glime the question (question mark not necessary), followed by an asterisk, then the answer (and any other answers, also seperated by an asterisk). Here are some examples:

/msg Glime TriviaTime add Planets: What is our planet called*Earth

For multiple answers:

/msg Glime TriviaTime add Presidents: What is the name of our president*Barack Obama*Obama*President Obama

Alternatively, for KAOS questions:

/msg Glime TriviaTime add KAOS: What were the most popular names in 2012*Amanda*Adam*Kyle*Kylie*Jordan

Thanks for submitting the questions! They will enter into circulation once they are approved.

Thank-you for the help! It means a lot to all of us. Those who’ve edited at least 100 questions are eligible for an editor cloak. In future versions of TriviaTime, you will also receive a badge!
