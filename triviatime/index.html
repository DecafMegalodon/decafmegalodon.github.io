TriviaTime

TriviaTime is the future of #trivialand. It will be our own, exclusive trivia plugin that will replace our current setup. Instead, SupyMonkey will replace Glime with the plugin once the project is complete.

The project has begun! We will be using GitHub to allow other people to contribute to it. Check out our Concept page for information on the design of the new plugin. Feel free to submit a pull request with your code once you’ve made changes.

Do you know Python or PHP and want to get involved?

Those interested in getting involved should first familiarize themselves with creating supybot plugins (Python) or creating PHP websites. Join us in #trivialand-dev on Freenode when you’re ready to get started!
