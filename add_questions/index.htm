Add Questions

This page is currently outdated, as a new system has been put into place.

 

We’ve deleted a lot of bad questions recently. However, we would like to replace them so there are more questions to cycle and they don’t feel repetitive. If you’d like to help us add questions or KAOS, here’s how.

Simply message Glime “add” followed by the question and answer (separated by an asterisk). Like this:

/msg Glime add What is our planet called*Earth

Alternatively, for KAOS questions:

/msg Glime add KAOS: What were the most popular names in 2012*Amanda*Adam*Kyle*Kylie*Jordan*Emily*Max*Michael*Ashley*Carley

Thanks for the help! We really appreciate it!
